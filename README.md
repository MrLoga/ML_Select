ML_Select
=========

Custom Select

  

`$('#S1').ML_Select({`  
 `width: 80,` // Ширина select  
 `height: 350,` // Высота select  
 `oHeight: 40,` // Высота option  
 `transformTiming: 'ease',` // Тип анимации  
 `transformTime: 0.5,` // Время анимации  
 `middle  : false` // Если true, то выбор будет в центре  
`});`
